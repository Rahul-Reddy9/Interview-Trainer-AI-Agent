AI Interview Trainer Agent
Agentic AI Powered Mock Interview & Career Preparation System

This project is an AI-powered Interview Trainer Agent built using Agentic AI, IBM watsonx.ai, IBM Granite models, LangFlow, and RAG.
It helps students and job seekers prepare for interviews through resume analysis, mock interviews, AI evaluation, and personalized learning recommendations.

Problem Statement

Many students struggle with interview preparation due to lack of guidance, feedback, and structured practice.
This system solves that by providing an intelligent, adaptive, and personalized interview preparation platform.

Objective
Extract skills from resumes
Generate technical & HR interview questions
Evaluate user answers using AI
Identify skill gaps
Provide personalized learning paths
Support company-specific preparation
System Architecture

User Input → Resume Analysis Agent → Question Generator Agent → Evaluation Agent → Learning Recommendation Agent → IBM Granite Model → Output

Key Features
Resume-based skill extraction
AI-generated interview questions
Smart answer evaluation & scoring
Personalized learning recommendations
Company-specific interview preparation
kill gap analysis
Adaptive difficulty system

Technologies Used
LangFlow – Multi-agent workflow design
IBM watsonx.ai – AI model deployment & embeddings
IBM Granite 4.0-8B-Instruct – Language understanding & reasoning
RAG (Retrieval-Augmented Generation) – Context-aware responses
FAISS / Chroma – Vector database for semantic search
IBM Cloud Lite – Deployment & scalability

Multi-Agent System
Resume Analysis Agent – Extracts skills and profile data
Question Generator Agent – Creates technical & HR questions
Evaluation Agent – Scores and analyzes answers
Learning Recommendation Agent – Suggests improvement roadmap

Expected Outcomes
Improved interview performance
Better skill awareness
Personalized learning experience
Increased placement readiness
AI-driven career guidance

Future Scope

Placement prediction system
